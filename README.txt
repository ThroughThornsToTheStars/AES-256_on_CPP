Привет, это реализация AES-256 на C++

Для работы программы нужно запустить AES-256.exe

Файлы, появляющиеся при работе программы:

encrypted.bin - зашифрованный файл
decrypted.bin - расшифрованный файл в битах
output.txt - расшифрованный файл в txt формате. Txt будет при любом текстовом формате.

Описание файлов, в проекте:

TextToBin.cpp - переводит текст в битовый файл.
Encrypt-Decrypt.cpp - AES-256 реализация на C++. На вход подается бинарный файл. На выходе encrypted.bin и decrypted.bin
AES-256CPP.cpp - файл проекта, который нужно запускать. Совмещает в себе оба вышеприведенных файла. На выхое output.txt
